## Getting Started

- Clone this project
- Start by installing npm if you don't have them already.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tridevVerma/React-Chat-App.git
   ```
2. Install NPM packages
   ```sh
   npm install
   cd client/
   npm install
   ```
   
### Run Project

  ```sh
  npm start
  cd client/
  npm start
  ```
